The definite plural in Swedish

As this exercise shows, it is often possible to predict the plural form of a noun (going from singular to plural, as in "car" → "cars"). Once you have mastered that (i.e. the indefinite plural), it is time to learn the definite plural of the noun (as in "car" → "the cars"). There are three rules to remember:

Rule 1 – covers 75% of all verbs

If the plural ends in a vowel + -r (e.g. vägar, gator, flickor), add -na:

gator → gatorna [streets → the streets]

Note: Nouns ending in -are (e.g.lärare, hamburgare) first drop the -e:

lärare → lärarna [teachers → the teachers]

Rule 2

If the plural ends in any other consonant (i.e. not -r), add -en:

hus → husen [houses → the houses]

Note: Stems ending in -el, -en, -er first drop the -e:

fönster → fönstren [windows → the windows]

Rule 3 – only for ett-words

If the plural ends in a vowel + -n, (e.g. äpplen, yrken), add -a:

äpplen → äpplena [apples → the apples]

Apply these eight
prediction rules

For a very large portion of Swedish nouns (over 65 percent), the plural form can be predicted. Here you will become familiar with eight important tools. Keep an eye on:


  1. The gender of the noun
  2. Word stress (underlined)
  3. How the word ends

If you don't know what the word means, you only need to click on it.


The symbol | indicates "ends with".


Rules

  1. Final stress N |cons / T |vowel (N affär, T kafé).
    Solution: add -er.
  2. N gen. |unstressed -a (kvinna).
    Solution: drop -a, add -or.
  3. N gen. |unstressed -e (pojke).
    Solution: drop -e, add -ar.
  4. T gen. |vowel (äpple, minne).
    Solution: add -n.
  5. T gen. |consonant (hus, barn).
    Solution: no change (Ø).
  6. |are or |er (hamburgare, indier).
    Solution: no change (Ø).
  7. |(n)ing (tidning, samling).
    Solution: add -ar.
  8. |het (nyhet, hemlighet).
    Solution: add -er.
